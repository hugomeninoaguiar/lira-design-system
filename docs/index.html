Lira · Design System

One system. Three surfaces.

Shared primitives — color, type, space, radius, shadow — power three semantic themes: Mobile (Women), Web (Women), Web (Professionals). Use the switcher above to compare.

Mobile · Women

Daily support, warm + visual

Peach app background, soft mint accents, illustrated character, rounded everything. Built for low-overwhelm daily use.

Web · Women

Premium marketing, same warmth

Larger type ladder, refined spacing, peach hero, mint accents. Same warmth as the app — sized up for marketing.

Web · Professionals

Clinical, restrained, credible

Cool off-white surfaces, sage as the lead accent, tighter radii, serif display for editorial weight. DSM-5-TR-ready feel.
