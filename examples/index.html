Lira Design System

One design system for the four Lira surfaces. Tokens and themes flow from a single source through generated CSS and JS into the Women app, the marketing site, and the clinical product.

Women app — time of day buttons

Preview of semantic.time.* tokens used by Lira for Women check-in buttons.

Use the toggle above to swap between Women and Professionals themes. Every value below is a generated CSS variable.

Surfaces

Women

Mobile app + marketing

Warm, calm, slightly magical — never clinical. Pill buttons, soft shadows, generous spacing. Voice: first-person Lira.

Professionals

Clinical product + marketing

Mostly neutral clinical tool with subtle Lira DNA. Cooler palette, denser layouts, sage primary action. Voice: third-person, evidence-led.

Quick reference

Install

"@lira/design-system": "github:hugomeninoaguiar/lira-design-system#main"

Web (Tailwind v4) — globals.css

@import "@lira/design-system/tokens.css";
@import "@lira/design-system/tailwind";
@import "tailwindcss";

React Native — App root

import { LiraThemeProvider } from "@lira/design-system/react-native";
<LiraThemeProvider>...</LiraThemeProvider>

Set theme

// Web
<html data-theme="professionals">

// React Native
<LiraThemeProvider themeName="professionals">
