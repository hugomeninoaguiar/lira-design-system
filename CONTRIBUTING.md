# Contributing

See **[docs/09-contributing.md](./docs/09-contributing.md)** for the full guide.

Quick version:

- Tokens: edit `tokens/`, run `npm run build:tokens`, commit `tokens/` + `dist/`.
- Components: edit `src/react-native/components/`, never use raw hex.
- Docs: edit the matching file in `docs/`.
- Open a PR. Use one of the labels listed in the contributing doc.
